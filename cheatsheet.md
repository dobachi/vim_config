# vim cheetsheet

## vim

### buffer

:ls : list buffers
:b [tab] : chose the buffer

## scrooloose/nerdtree

`,n` : Open nerdtree

## simeji/winresizer

`<C-n>` : Change window size

### functions

" o: open in prev window
" t: open in new tab

## unite-outline

`,u` : Open outline

## Deol (Terminal)

`,u` : Open a terminal with splited window

`:Deol`; Start a terminal in the current window

`<C-w> N`: Text mode

`i`: Enter normal terminal mode
